# MLIR

MLIR (Multi-Level Intermediate Representation) is the backbone of the modern ML compiler stack — Google's IREE, Apple's silicon stack, AMD ROCm, and torch-mlir all speak it. Its core idea: don't try to represent everything at one level. Have many *dialects*, lower from one to the next, and run optimizations at the level where they're cheapest.
