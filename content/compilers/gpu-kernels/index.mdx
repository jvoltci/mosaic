# GPU Kernels

ML compilers cover ~90% of cases. The remaining 10% — where you need a custom attention variant, a fused op the compiler can't see, or a kernel for a new chip — is hand-written. This module covers the modern kernel-writing tools: Triton on the server side, and Vulkan / Metal Performance Shaders on mobile and desktop GPUs.
