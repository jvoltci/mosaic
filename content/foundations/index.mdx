## Modules in this track

- **[C++ Memory Model](/foundations/cpp-memory)** — stack vs heap, RAII, move semantics, smart pointers
- **[Caches & Hardware Reality](/foundations/caches-and-architecture)** — cache lines, branch prediction, data-oriented design
- **[Parallelism & SIMD](/foundations/parallelism)** — threads, lock-free programming, vector intrinsics

## What you'll be able to do after

- Read C++ code and predict where it allocates, where it copies, and where it stalls
- Recognize and fix cache-unfriendly memory access patterns
- Reason about why a "trivial" change to a loop can be 10× faster
- Have the vocabulary to read a CPU profiler output without getting lost
