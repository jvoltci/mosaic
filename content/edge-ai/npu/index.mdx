# NPU Stacks

NPUs (Neural Processing Units) live alongside the CPU and GPU on every modern phone SoC. Each vendor ships its own programming model — Apple ANE via Core ML (covered in [On-Device Runtimes](../on-device)), Qualcomm Hexagon via QNN, MediaTek Genio, AMD XDNA. This module covers the Qualcomm side; Apple's is in [Core ML & ANE](../on-device/coreml).
