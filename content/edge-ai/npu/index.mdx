# NPU Stacks

NPUs (Neural Processing Units) live alongside the CPU and GPU on every modern phone SoC. Each vendor ships its own programming model — **Qualcomm Hexagon** via QNN (Snapdragon phones, robots, drones), **Apple Neural Engine** via Core ML (every iPhone since 2017), MediaTek APU, AMD XDNA, ARM Ethos. This module covers the two stacks that ship the most production inference today: Hexagon and ANE.
