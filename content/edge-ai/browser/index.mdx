# Browser (WebGPU)

The browser is the universal sandbox: every phone, every laptop, every Chromebook. WebGPU (stable in Chrome 113+, Safari 26+, Edge 113+) makes it real — you can compile a transformer kernel to WGSL once and have a 7B model run on a kid's $200 laptop with no install. This module covers the runtime stack and the practical limits.
